## 우선순위 기준

| 등급 | 설명 | 기준 |
|------|------|------|
| P0 | 필수 | 핵심 기능, 장애 시 서비스 불가 |
| P1 | 높음 | 주요 기능, 사용자 경험에 영향 |
| P2 | 보통 | 부가 기능, 있으면 좋음 |
| P3 | 낮음 | 엣지 케이스, 나중에 해도 됨 |

## 상태 범례

- ⬜ 대기: 아직 작성 안 함
- 🔄 진행: 작성 중
- ✅ 완료: 테스트 통과
- ❌ 실패: 테스트 실패 (수정 필요)
- ⏸️ 보류: 블로커로 인해 중단
